# nvim-dotfiles-lua

## Package required:
- neovim 0.7.0+
- ripgrep (For Telescope plugin)
- A Nerd fonts (For displaying icon of file, symbol, etc)
- Nodejs >= 14.4

## Plugins are used
- packer.nvim
- nvim-tree.lua
- nvim-web-devicons
- alpha-nvim
- lualine.nvim
- sonokai
- telescope.nvim
- indent-blankline.nvim
- Comment.nvim
- vim-fugitive
- gv.vim
- nvim-treesitter
- coc.nvim

## Mapping

| Mode       |    Key        | Function |
| ---------- |:-------------:| -----|
| Normal     | \<Leader\>\<Up\> | Move to above split window |
